Instruction Set
COMMANDS:
  NONE: do nothing
  ADD: C = A + B
  SUB: C = A - B
  AND: C = A & B
  NOT: C = ~A
  EQ:  C = 11111111 or 00000000
  BRANCHEQ: if C address  = 11111111 branch else do nothing
  LOAD: LOAD MEM(a+b) into REG(c)
  STORE: Store REg(c) into MEM(a+b)

INSTR BIT REQ: 14 bit instructions
  A ADDRESS: 3 BIT
  B ADDRESS: 3 BIT
  C ADDRESS: 3 BIT
  OPCODE:    3 BIT
  BR ADDR:   A + B ADDRESS
  MEM ADDR:  A + B ADDRESS
  REG ADDR:  C ADDRESS

LAYOUT:
ADD/SUB/AND/NOT [OP][C][A][B]
BRANCH [OP][EQ][BR]
MEM [OP][REG][MEM]

INTRUCTION REGISTER
  64 12 BIT ARRAYS
MEMORY
  64 8 BIT ARRAYS
REGFILES
  8 8 BIT ARRAYS
